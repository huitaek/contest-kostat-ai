# <b>2022 통계데이터 인공지능 활용대회</b>

## <b>“자연어 기반 인공지능 산업분류 자동화”</b>

자연어 기반의 통계데이터를 인공지능으로 자동 분류하는 기계학습 모델 발굴로 통계 데이터 활용 저변 확대

### <b>공모분야</b>
산업분류 자동화 인공지능 모델 개발

### <b>필수 분석 조건</b>
Python 또는 R 언어로 자연어 텍스트 마이닝 및 인공지능 분류 모델 구축

### <b>제출물</b>
- 산업분류 예측값 (csv파일)
- 작성 코드
- 코드실행 로그
- 코드설명자료 (이메일제출, 2페이지 내, 2차 심사 대상자)

<b> 활용 데이터 </b>
전국사업체조사 샘플데이터
